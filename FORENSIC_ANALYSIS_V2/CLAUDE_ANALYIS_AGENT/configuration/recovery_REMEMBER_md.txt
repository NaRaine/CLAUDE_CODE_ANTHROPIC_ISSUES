## IMPORTANT PATTERNS

### When Starting a Session
1. Read `/media/raine/VM/CLAUDE_START_UP/01_INIT/CLAUDE.md`
2. Check RaineWare orchestrator is running
3. Check for active tasks in `04_TASKS/active/`
4. Review last session status
5. Set terminal background to black (shows you've read instructions)

### When Ending a Session
1. Save session status to `05_SESSIONS/LAST_SESSION.md`
2. Document any incomplete tasks in `04_TASKS/active/`
3. Commit any code changes to git
4. Note what you were working on

### When Detecting a Crash
1. Check logs for errors
2. Kill hung processes if needed
3. Remove stale lock files
4. Restart orchestrator
5. Log recovery action to `06_RECOVERY/crash_log.txt`
